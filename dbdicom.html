open(path=None, unzip=False)[source]

Opens a DICOM folder for read and write.

Reads the contents of the folder and summarises all DICOM files in a dataframe for faster access next time. The dataframe is saved as a pkl file when the folder is closed with .close(). All non-DICOM files in the folder are ignored.

Parameters

path – The full path to the directory that is to be opened.
