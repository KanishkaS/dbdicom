dbdicom.ds.dataset.set_values(ds, tags, values, VR=None)[source]

Sets DICOM tags in the pydicom dataset in memory

Private and standard tags can both be set. tags, values and VR must either be lists of equal lengths, or single values. VR is required for private tags. If private and standard tags are set in the same function call, VR can be set to any value for the standard tags: e.g.

set_values(ds, [‘Rows’, (0x0019, 0x0100)], [128, ‘Hello’], [None, ‘LO’])
